# RAG 数字人 LLM 智能问答评测系统

## 项目简介
基于知识图谱 + RAG 检索增强的本地大模型智能问答系统，配套自动化评测工具，用于量化评估大模型回答效果，解决纯大模型输出幻觉、无依据等问题。

## 技术栈
- 编程语言：Python
- 知识库：Neo4j 图数据库
- 大模型：Deepseek（本地部署）
- 检索增强：RAG
- 数字人：Stable Diffusion
- 数据处理：Pandas、NumPy

## 核心功能
1. **知识图谱构建**：使用 Neo4j 完成专业课程知识点实体、关系建模与查询优化
2. **RAG 问答链路**：本地部署 Deepseek 大模型，搭建检索增强问答流程
3. **LLM 自动化评测**：自研 Python 批量评测工具，支持幻觉率、知识匹配准确率、指令遵循度、响应时延等 16 项量化指标
4. **数字人交互**：基于 Stable Diffusion 生成教学数字人形象，完成交互调试

## 项目成果
- 自动化评测工具将人工校验工作量降低 65%，百条样本单轮评测耗时由 30 分钟缩短至 2 分钟
- RAG 策略优化后，问答知识匹配准确率提升 42%，大模型幻觉出现频次下降 51%
- 产出可复用 LLM 自动化评测脚本与标准化指标体系
